## How to setup

### Requirements

- Docker Compose

### Steps

Run `docker-compose build` in project root folder.

## Running specs

Run `docker-compose run --rm web bash`. This command will open a bash in project
folder and run rspec.

## Running app

Run `docker-compose up`.

## Architecture's Components

- Rails ~> 5.0.0
- Postgres
